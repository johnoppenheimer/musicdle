## Getting Started

This project use [Gazr](https://gazr.io/) to manage command.

To install the dependencies:
```bash
make init
```

To run the development server:
```bash
make watch
```

To check the linter:
```bash
make style
```

To fix the issue with the linter
```bash
make format
```
